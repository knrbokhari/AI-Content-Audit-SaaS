# AI Content Audit SaaS

This is a full stack SaaS Project. A SaaS where website owners paste their website URL and get:

- SEO audit
- Content quality score
- AI-generated improvements
- Keyword suggestions
- Competitor analysis
- Content rewriting

## Tech Stack

- Frontend:
    - Next.js
    - TypeScript
    - Tailwind
    - Shadcn

- Backend:
    - NestJS
    - PostgreSQL
    - Prisma

- Payments:
    - Stripe Subscription

- AI:
    - OpenAI API

## Feature

- Role based access control
- Subscription System with Stripe,
- AI-powered website content audits
- Invite Team Members
- Customized theme for each customer
- White-label Reports 
- AI Content Generation
- AI Chat Assistant
- Scheduled Audits
- Competitor Analysis
